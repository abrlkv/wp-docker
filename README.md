## About wp-docker:

Deploying wordpress project with docker.

## How to use:
1. Edit config in run.sh
2. Run `./run.sh [arg]`

Args:
- 'start' - runing docker
- 'stop' - stoping docker
- 'b' - creating wp-content and db backup
- 'bf' - creating full project backup
- 'd' - creating db dump
- 'i' - import db dump
- 'deploy' - deploy project on test server
