# wp-docker
##About:

Deploying new wordpress project with docker.

##How to use:
1. Edit workpath in run.sh
2. Run `./run.sh [arg] [projectname]`

Args:
- 'new' - create new project
- 'start' - runing docker
- 'stop' - stoping docker
- 'b' - creating wp-cotent and db backup
- 'bf' - creating full project backup
- 'd' - creating db dump
- 'i' - import db dump
